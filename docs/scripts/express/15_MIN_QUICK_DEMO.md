# 15-Minute Quick Demo Script

**Purpose**: Show core YearWheel value in a tight timeframe  
**Duration**: 12-15 minutes + 3-5 min Q&A  
**Best for**: Initial discovery calls, busy decision makers, time-constrained demos  
**Goal**: Get them excited enough to book a longer session or start trial

---

## Pre-Demo Setup (2 minutes before call)

### Prepare Demo Environment
- [ ] Open YearWheel in clean browser window (no other tabs visible)
- [ ] Use demo account with ONE pre-built wheel (marketing/HR based on lead)
- [ ] Zoom set to 100% (full year view) - use Zoom tab to select "Full year"
- [ ] Left sidebar open to Disc view (they need to see the structure)
- [ ] Have blank wheel ready in another tab (for quick creation demo)

### Know Your Audience
- What industry? (Marketing, HR, Education, Project Management)
- Current pain point? (Spreadsheets, Gantt charts, no visualization)
- Team size? (Determines collaboration emphasis)

---

**Say while doing this:**

### Step 2: Create Activity Groups (30 seconds)
**Scroll down to Activity Groups section â†’ Click "+ Add Activity Group"**

**Create 4 groups quickly:**
1. Name: "Q1", pick blue color, click Add

## Q&A - Common Questions & Answers

### "How much does it cost?"
**Answer:**
> "YearWheel starts free - you can create 2 wheels and have up to 3 team members. If you need more wheels or unlimited team members, Premium is [price] per month. Most teams start free and upgrade as they grow."

**Then:**
> "Would you like to start with a free trial today? I can send you the link right after this call."

### "Can we import from [Excel/Google Sheets/Asana]?"
**Answer:**
> "You can export data from [tool] and manually input into YearWheel, which based on what we just saw takes about 10-15 minutes for a full year. We're working on direct imports for our Premium tier."

**If they push:**
> "Would this be a blocker? If you have a file, I can help you map it out in a follow-up session."

### "What if we need more complex dependencies/Gantt features?"
**Answer:**
> "YearWheel is designed for high-level annual planning and visualization - not task-level project management. Think of it as complementary to tools like Asana or Monday. You use YearWheel for strategic planning and those tools for day-to-day execution."

**Qualify:**
> "Is your main need strategic annual planning, or detailed task management?"

### "Can we customize colors/branding?"
**Answer:**
> "Yes - every activity group has customizable colors, and when you export, you can adjust in design tools if needed. We're adding brand theme support to Premium soon."

### "Does it work on mobile?"
**Answer:**
> "YearWheel works on tablets (iPad works great). Mobile phones are view-only currently - we find most planning happens on desktop. Is mobile editing critical for your team?"

### "How secure is our data?"
**Answer:**
> "All data is encrypted in transit and at rest, hosted on Supabase (built on AWS). We're SOC 2 Type II compliant [if true], and you can export your data anytime. Want me to send you our security overview?"
