## Image Guidelines

- **Format:** PNG with transparency where appropriate
- **Quality:** High-resolution (minimum 1200px width for main screenshots)
- **File size:** Optimize images to keep under 500KB each
- **Annotation:** Consider adding arrows or highlights to draw attention to key features
- **Branding:** Ensure screenshots show current YearWheel branding and colors

## Optimization

Before adding images, optimize them using:
- [TinyPNG](https://tinypng.com/) for PNG compression
- [Squoosh](https://squoosh.app/) for advanced optimization

## Fallback

If screenshots are not yet available, the guide will still display with text content only. The image elements are designed to degrade gracefully.
