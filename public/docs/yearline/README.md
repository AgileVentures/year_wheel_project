# YearLine How-To Guide Screenshots

This directory contains screenshots for the YearLine (Monday.com timeline) integration how-to guide.

## Required Images

Save the provided screenshots with these exact filenames:

1. **01-timeline-overview.png** - Main timeline view showing the YearLine interface with groups and colored task bars across months
2. **02-color-themes.png** - Color theme dropdown selector showing available themes (Monday Colors, Pastel Dreams, Vibrant Energy, etc.)
3. **03-edit-item.png** - Edit dialog showing task detail fields (Name, Group, Assigned To, Status, Start Date, End Date)
4. **04-week-view.png** - Timeline zoomed to week view showing ISO week numbers (W44, W45, W46, etc.) in the header
5. **05-status-filter.png** - Timeline filtered by status showing grouped view with different status categories

## Image Guidelines

- **Format**: PNG preferred for crisp screenshots
- **Resolution**: Full width screenshots, ideally 1920px or higher for clarity
- **Content**: Should show real YearLine interface from Monday.com app
- **File naming**: Use exact filenames listed above (lowercase, dash-separated)

## Usage

These images are referenced in `/src/pages/YearLineHowToGuide.jsx` and displayed in the step-by-step guide at `/yearline/how-to`.

The guide is designed to be iframe-embeddable in the Monday.com marketplace.
